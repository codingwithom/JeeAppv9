That looks like a very sleek, modern Time Management Dashboard. To build something like this, you’ll need a prompt that covers the specific UI elements (dark mode, cards, modals) and the functional logic (countdown, lap timing, alarm scheduling).

Here is a comprehensive prompt you can use with an AI (like Gemini) to generate the code for this system:

Prompt for AI Code Generation
"Build a modern, dark-themed Time Management Web System using HTML, CSS, and JavaScript (or a framework like React). The UI should be a clean dashboard with three main sections: Timers, Alarms, and a Stopwatch.

1. General Design Specs
Theme: Dark mode (charcoal background, light gray text, vibrant blue for active buttons).

Layout: Responsive card-based grid. Use sleek icons (Play, Pause, Reset, Flag, Trash).

Modals: All editing should happen in a centered popup modal with a blurred background overlay.

2. Timer System
Grid View: Multiple cards showing the time remaining, a circular progress ring, and Play/Pause and Reset buttons.

Edit Modal: When clicked, open a modal with:

Up/Down arrow selectors for Hours, Minutes, and Seconds.

A text input field for a custom 'Timer Name'.

Save and Cancel buttons.

Logic: Multiple timers should be able to run simultaneously.

3. Alarm System
Card View: Shows the alarm time (e.g., 8:30 AM), a 'Time until' countdown (e.g., 'In 12 hours'), a label, and active days of the week.

Edit Modal:

Time picker with AM/PM toggle.

'Repeat Alarm' checkbox with a circular selector for Sunday–Saturday.

Dropdowns for 'Alarm Sound' and 'Snooze Duration' (5, 10, 20 mins).

Save button that adds the alarm to the dashboard.

4. Stopwatch System
Display: Large digital readout showing 00:00:00.00 (Hours:Minutes:Seconds.Centiseconds).

Controls: A main Play/Pause button, a Flag (Lap) button, and a Reset button.

Lap List: Below the timer, a scrollable table with three columns: 'Lap #', 'Lap Time', and 'Total Time'.

5. Functional Requirements
The system should be 'stateful' (timers keep running even if I switch tabs).

Include a notification or sound alert when a timer or alarm finishes."

Tips for your Website Implementation:
Circular Progress: Use an SVG for the progress ring around the timers. It’s the smoothest way to animate a countdown visually.

Local Storage: Tell the AI to use localStorage so that when you refresh your website, your set alarms and timers don't disappear.

Sounds: You’ll need to host small .mp3 files (like a "Chime" or "Jingle") to make the alarm selection dropdown actually work.
